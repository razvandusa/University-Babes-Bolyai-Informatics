# Client-Server Concurent

## 📌 Cerințe

### 1️⃣ Rulare pe calculatoare diferite
- Rulați o pereche de programe client-server pe calculatoare diferite.
- Modificați clientul astfel încât să se poată conecta la orice server (nu neapărat la serverul care rulează pe localhost – 127.0.0.1).
- Serverul trebuie specificat prin adresă IP și port în linia de comandă a clientului.

Exemplu:
```bash
./client <ip_server> <port>
